# bytes
Utility to parse a string bytes (ex: 1TB) to bytes (1099511627776) and vice-versa.</br>
This is a 1:1 recration of the [bytes](https://www.npmjs.com/package/bytes) package on [npm](https://www.npmjs.com/)
# Installation
Get this module from [lit](https://luvit.io/lit.html) with the install command
```
lit install TohruMKDM/bytes
```
# Usage
As this is a 1:1 recreation you should be able to refer to the original package's readme for tips on how to use this module </br>
https://github.com/visionmedia/bytes.js/blob/master/Readme.md </br>
If something doesn't work as expected or if the io does not match the npm package then please open an issue.
